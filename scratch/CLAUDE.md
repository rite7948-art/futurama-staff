# CLAUDE.md — scratch/

Папка одноразовых отладочных/исследовательских скриптов. **НЕ часть продакшена** — сюда
складываются временные утилиты для проверки данных, разбора Google Sheets CSV, отладки
голосовой статистики и проверки БД.

## Категории файлов
- `inspect_*.php`, `inspect_*.js` — разбор структуры Google Sheets / CSV (колонки, GID, листы).
- `check_*.php`, `check_*.js` — проверки БД, голосовых сессий, схемы, дат, колонок.
- `debug_*.php`, `debug_*.js` — точечная отладка (например, пересечения смен, роли, данные юзеров).
- `test_*.php` — тесты вебхука Apps Script (`test_webhook*.php`), добавления саппорта, очереди.
- `verify_*` — верификация исправлений и ID.
- `selfbot_sync.js`, `sync_checker.js` — ранние прототипы синхронизации (актуальная логика
  в корневом `check_sync.js`).
- `*.csv` — выгрузки таблиц для локального анализа.

## Правила
- Ничего отсюда не подключается из боевого кода. Можно безопасно удалять.
- При создании новых временных скриптов для разбора данных — класть сюда, не в корень.
- Секреты не хардкодить; брать из `app_config.php` / env.
